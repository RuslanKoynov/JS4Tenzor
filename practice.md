### Упражнение 1: Базовое понимание promise
Напиши функцию delay(ms), которая возвращает promise, разрешающийся через указанное количество миллисекунд. Используй эту функцию, чтобы вывести в консоль сообщение "Прошло 2 секунды" после задержки в 2000 мс.

Пример использования:

```javascript
delay(2000).then(() => console.log('Прошло 2 секунды'));
```
*Подсказка*: Внутри промиса используй setTimeout, который вызывает resolve по истечении времени.

### Упражнение 2: Обработка успеха и ошибки
Создай promise, который имитирует проверку пароля. Пусть он завершается успешно (вызывает resolve), если переданная строка равна "qwerty", и завершается с ошибкой (вызывает reject) в противном случае. Напиши код, который обрабатывает успешное выполнение и ошибку, выводя соответствующие сообщения в консоль.

Пример использования:

```javascript
checkPassword('qwerty')
  .then(() => console.log('Пароль верный'))
  .catch(() => console.error('Пароль неверный'));
```
*Подсказка*: Функция checkPassword должна возвращать новый промис, в котором проверяется условие.

### Упражнение 3: Цепочка промисов
Напиши функцию step1, которая возвращает promise, возвращающий число 5 через 1 секунду. Затем напиши функцию step2, которая принимает число, умножает его на 2 и возвращает promise, который возвращает результат через 1 секунду. Создай цепочку promise, которая сначала вызывает step1, затем передает результат в step2 и выводит окончательный результат (который должен быть 10) в консоль.

Примерный скелет:

```javascript
function step1() {
  return new Promise((resolve) => {
    setTimeout(() => resolve(5), 1000);
  });
}

function step2(number) {
  // Твоя реализация
}

// Цепочка вызовов .then()
step1()
  .then(/* передай результат в step2 */)
  .then(/* выведи финальный результат */);
```

### Упражнение 4: Преобразование колбэков в promise
Дан следующий код, использующий колбэки:

```javascript
function loadData(callback) {
  setTimeout(() => {
    const data = { id: 1, name: 'John' };
    callback(data);
  }, 1000);
}
```
Перепиши функцию loadData, чтобы она возвращала promise. Затем используй этот promise с помощью async/await для вывода данных в консоль.

*Требования*:
- Создай новую функцию loadDataPromise, которая возвращает promise.
- Напиши асинхронную функцию, которая использует await для получения данных из loadDataPromise и выводит их в консоль.
